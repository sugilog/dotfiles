About
------------------------------------------------------------
This repository manage the sugilog's individual settings.

HowToUse (repository)
------------------------------------------------------------
### Init
- clone this repository.
- pull and update submodules
```
git submodule init
git submodule update
```

### Update (sometimes)
- better to update submodules.
```
git submodule foreach 'git pull origin master'
git submodule update
```
